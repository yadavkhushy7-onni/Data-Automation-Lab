# Data & Automation Lab

A modern personal portfolio website for AI automation, Python automation, data analysis, Power BI dashboards, and workflow automation services.

## Files

- `index.html` - main site structure and content
- `styles.css` - responsive styling and animations
- `script.js` - navigation, reveal effects, and contact form UI
- `serve.ps1` - lightweight local static server for Windows PowerShell

## Run locally

Open `index.html` directly in a browser, or run:

```powershell
powershell -ExecutionPolicy Bypass -File .\serve.ps1
```

Then visit `http://127.0.0.1:8000`.

## Deployment

This is a static site and can be deployed for free on:

- Netlify
- GitHub Pages
- Vercel
